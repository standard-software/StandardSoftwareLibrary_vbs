■概要
このプログラムソースコードライブラリは
[Standard Software Library For VBScript]です。

Windows PC で動く VBScript の実用的な関数群です。

言うまでもありませんが、
VBScriptはBatファイルに変わる高機能なプログラミング言語です。
テキストエディタがあれば、Windows PC 上で簡単に
プログラムを作成できるために非常に利用者が多い言語です。

手軽に高機能な言語ですので、元々から開発効率は高いですが
VBScriptでの開発をさらに効率のよいものにするために
VBScriptの標準にはないが汎用的に使える機能を関数としてまとめて
オープンソースのライブラリとして
スタンダードソフトウェアブランドとして公開させていただきます。

スタンダードソフトウェアでは
ソースコードライブラリを提供し
開発者の方がより素早く望む機能を実現させる事ができるように
支援させていただきます。

より多くの開発者の方によいものを提供させていただくために
開発資金を得る必要もあり、また、
他の方の開発の協力もお願いしたいと考えておりますので
ライセンスは GPL と 商用 のデュアルライセンスとさせていただきます。

よろしくお願いいたします。

このプログラムソースコードのオープンソースライセンスは
GPL または、商用ライセンスのデュアルライセンスです。

このプログラムを使用の際に、GPLに準拠して
オープンソースとして使用することができます。

GPLの厳密な詳細については
インターネットで検索をお願いいたします。

このライブラリを自分の作成したプログラムを他の人に配付する際に
プログラムソースを公開したくない場合。
GPLに準拠できなくなりますので、
その際には、商用ライセンスをご購入ください。

■ライセンス選択
Standard Software Library For VBScript
を用いてプログラムを開発する時、

・  個人で使用するプログラムの開発をする場合。
    
    プログラムの配付がありませんので
    商用ライセンスは一切必要ありません。
    また、GPL にも触れませんので、
    他の方にソースコードを公開する必要はありません。

・  組織内で使用するためのプログラムの開発をする場合

    個人で使用する場合と同様に
    プログラムの配付がありませんので
    商用ライセンスは一切必要ありません。
    組織外の方にソースコードを公開する必要はありません。
    組織内の他の方はプログラムソースコードを
    見ることが出来ると想定でき、
    これは GPL に準拠します。

・  個人として他の人に対して、また
    組織として他の組織に対して、
    プログラムを開発し提供する場合。

    ・  プログラムの配付相手にソースコードを公開してもよい場合。

        商用ライセンスは必要ありません。

    ・  プログラムの配付相手にソースコードを公開したくない場合。

        商用ライセンスが必要です。

■ケース

・  Aさんは、Bさんにプログラムを渡して
    ソースコードも配付した。

    このような場合は商用ライセンスは必要ありません。

・  Aさんは、Bさんにプログラムを渡して
    Bさんはソースコードを読める人ではなかったので
	ソースコードは渡さなかった。

    このような場合は商用ライセンスは必要ありません。

	Bさんから要望があった場合にソースコードを
	配付可能なように対応ください。
	また、GPLに基づいているとの事は
	Readmeやマニュアルにて明示ください。

・  Aさんは、プログラムを作成してインターネットで公開した。
    プログラムソースコードも配付対象にしている。
    
    このような場合は商用ライセンスは必要ありません。

・  Aさんは、プログラムを作成してインターネットで公開した。
    特定の自分の作成した部分は公開したくなかった。

    このような場合は商用ライセンスは必要です。

・  A社は社内だけで使うプログラムについて
    B社に発注する。
    B社はA社にソースコードを含めた納品を行う。

    このような場合はGPLに準拠しているので
    商用ライセンスは必要ありません。

・  A社は、一般向けに販売するソフトウェアについて
    B社に発注する。
    B社はA社にソースコードを含めた納品を行う。

    このような場合、一般向けに販売するソフトウェアが
    ソースコードを含めた全てを相手に公開する事は通常ないと
	考えられますので、商用ライセンスをご購入ください。

    ※GPLで公開する必要があるのは
      ライブラリを使用した全ての派生物ですので
      相手に対して、
      Standard Software Library For VBScript
      だけではなく、関連する全てのソースコードを
      公開する義務が発生してしまいますので
      ご注意ください。
